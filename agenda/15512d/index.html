#軟體開發

如何從零到千星？開源專案成功的實戰分享

schedule

14:25 - 14:35

location_on

R0

先備知識

對於 GitHub 以及開源有基本的認識與瞭解

議程介紹　

夢想讓自己的開源專案在 GitHub 上突破千星？這場演講將帶你瞭解如何成功打造好的開源專案，讓專案獲得廣泛關注的完整路徑。我會以我擁有 10k stars 的專案 superfile 爲例，描述如何打造一個具吸引力的開源專案，並結合實戰經驗，分享獲取社群支持的核心策略。

演講內容將涵蓋專案啟動的初期思考、面對挑戰時的應對方式，以及如何透過行銷、協作與技術實力，還有後續的維護，將一個普通專案轉變成千星級別的專案。讓自己的開源專案發光發熱！

yorukot

我是一名高二學生，對於將程式開發轉化為有意義的實作專案充滿熱情。相比競賽程式，我更喜歡專注於實際應用，創造能解決問題或帶來價值的工具。目前，我的主要開源專案爲 superfile ，一個專注於 UI 的檔案處理工具，在 GitHub 上獲得超過 10k stars。我期待未來能夠持續發展更多實用的專案，並透過開源的方式與更多人分享我的創作！
